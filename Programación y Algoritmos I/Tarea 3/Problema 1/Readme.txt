El programa recibe un parámetro, que es el nombre del archivo
que contiene los ejemplos a probar. Cada ejemplo se compone probar
un único entero, y los ejemplos deben estar separados por un salto 
de línea. El archivo "ejemplos.txt" contiene una entrada válida.

Para cada ejemplo de entrada, el programa imprime lo siguiente

El numero X es (no es) palindromo

para el caso en el que X sea o no sea palindromo.
